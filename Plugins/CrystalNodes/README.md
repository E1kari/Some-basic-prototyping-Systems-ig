# CrystalNodes
*By Skylake*

![UE5.2](https://img.shields.io/badge/UE-5.2-blue) to ![UE5.7](https://img.shields.io/badge/UE-5.7-blue)

A tiny plugin that allows you replace the brush of nodes by animated custom materials.

## Settings ##
➡️Editor preferences➡️Plugins➡️Crystal Nodes
![CrystalNodesSettings](https://imgur.com/gEPCgyi.jpg)

### Setup the Background ###
Click the button to replace editor grids to custom dot grids.

### Cursor Distance Field ###
Light effect related to the distance to cursor.
![Diff-CursorDF](https://imgur.com/2yg1izo.png)

### Point Light ###
Put a point light on the cursor, illuminate the nodes dynamicly when cursor move.
![Diff-PointLight](https://imgur.com/yMbJrSd.png)

### Outline Width and Expansion ###
Configurable distance scaled outline width.
![Diff-Width](https://imgur.com/R4RvGrD.png)
